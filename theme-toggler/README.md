# Dark Mode Toggler App

A simple dark mode toggler built with **React**, **Vite**, and **Tailwind CSS**. This app allows users to toggle between light and dark modes for a better user experience depending on their preference.

## Features

- Toggle between light and dark themes.
- Lightweight and fast due to Vite's build system.
- Styled using Tailwind CSS for easy customization.
- Simple and clean UI design.

## Technologies Used

- **React**: JavaScript library for building user interfaces.
- **Vite**: Next-generation front-end tooling to improve build performance.
- **Tailwind CSS**: Utility-first CSS framework for styling.
